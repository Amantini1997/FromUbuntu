Queue

The queue is a data structure of the type FIFO. It is composed of 3 methods:
  • Eqnueue, which add an element to the rear of the collection;
  • Pop, which removes and returns the element at the front of the queue;
  • Top, which returns the element at the front of the queue without removing it.
Beside these methods, it also possible the length property which, as the name suggests, returns the number of items in the queue.
